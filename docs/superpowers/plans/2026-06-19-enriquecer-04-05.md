### Task 1: Enriquecer Módulo 04 — Operadores

**Files:** Modify: `licoes/04-operadores.html`

**Design:**
- **Analogia (⚡):** operadores de comparação e lógicos são como os *intertravamentos de segurança* (interlocks) de uma máquina: o motor só liga se a proteção estiver fechada **E** a energia presente. O programa combina condições com `and`/`or`/`not` para decidir.
- **Ilustração SVG:** uma "porta lógica E (AND)" simples — duas entradas (`proteção fechada`, `energia ligada`) entrando num bloco `E` que produz uma saída (`motor pode ligar`). Theme-aware, com setas (`<defs>`).
- **Na engenharia (3 cards):** cada card imprime um booleano de decisão (sem `if`):
  - ⚙️ Mecânica: medida dentro da tolerância — `medida = 10.05` / `print("Dentro?", medida >= 9.8 and medida <= 10.2)`.
  - ⚡ Elétrica: tensão na faixa segura — `tensao = 220` / `print("Seguro?", tensao >= 210 and tensao <= 230)`.
  - 📡 Telecom: sinal acima do limiar — `sinal = -65` / `print("Link OK?", sinal > -70)`.

### Task 2: Enriquecer Módulo 05 — Condicionais

**Files:** Modify: `licoes/05-condicionais.html`

**Design:**
- **Analogia (⚙️):** uma condicional é uma *decisão automática de controle*: como um sistema que para a máquina **se** a temperatura passar do limite, ou um relé que desliga **se** a corrente for alta demais. O `if` escolhe a ação conforme a leitura.
- **Ilustração SVG:** um fluxograma com um **losango de decisão** (`condição?`) e dois caminhos rotulados `V` (verdadeiro → ação A) e `F` (falso → ação B). Theme-aware, com setas (`<defs>`).
- **Na engenharia (3 cards):** cada card usa `if`/`else` (ou `elif`):
  - ⚙️ Mecânica: classificar uma medida — `medida = 10.3` / `if medida > 10.2:` / `    print("Acima da tolerância")` / `else:` / `    print("OK")`.
  - ⚡ Elétrica: decidir ação pela tensão — `tensao = 245` / `if tensao > 250:` / `    print("Desligar!")` / `elif tensao > 230:` / `    print("Alerta")` / `else:` / `    print("Normal")`.
  - 📡 Telecom: qualidade do sinal — `sinal = -80` / `if sinal > -60:` / `    print("Excelente")` / `else:` / `    print("Fraco")`.
