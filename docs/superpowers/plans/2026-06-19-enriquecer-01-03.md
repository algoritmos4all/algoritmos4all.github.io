### Task 1: Enriquecer Módulo 01 — O que é um algoritmo

**Files:** Modify: `licoes/01-o-que-e-algoritmo.html`

**Interfaces:** Consumes molde 06; CSS didático; playground (matplotlib só no teaser). Preserva todo o conteúdo atual do arquivo.

**Design:**
- **Analogia (⚙️):** um algoritmo é uma *sequência de montagem* — como a ordem exata de etapas para montar uma peça na linha de produção, ou a rotina de partida de um motor: a ordem importa e cada passo precisa ser preciso. (Conecta a "receita" já existente ao mundo da engenharia.)
- **Ilustração SVG:** um fluxograma simples em sequência — `início → passo 1 → passo 2 → passo 3 → fim` com setas (mostra "ordem importa"). Usar `<defs>` de seta. Theme-aware.
- **Na engenharia (3 cards):** cada card descreve um algoritmo da área como passos `print` (só `print` é conhecido):
  - ⚙️ Mecânica: imprimir a sequência de partida de uma máquina (3–4 `print` em ordem).
  - ⚡ Elétrica: imprimir os passos para ligar um circuito com segurança.
  - 📡 Telecom: imprimir os passos para enviar uma mensagem (codificar → transmitir → confirmar).
  Ex. card ⚙️: `print("1. Travar a peça")` / `print("2. Ligar o motor")` / `print("3. Iniciar o corte")`.
