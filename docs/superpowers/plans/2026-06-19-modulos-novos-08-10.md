**Design do módulo:**
- **Analogia (⚙️):** uma função é uma *máquina/peça reutilizável* — entram parâmetros (eixo de entrada), ela processa, e sai um valor (`return`). Como uma caixa de engrenagens: você usa sem reabrir por dentro toda vez.
- **Ilustração SVG:** uma "caixa-função" rotulada `def f(x)` com setas de entrada (parâmetros) à esquerda e uma seta de saída (`return`) à direita; legenda explicando entrada → processamento → retorno.
- **Explicação progressiva (com exemplos executáveis):** (a) `def` + `return` com um exemplo mínimo; (b) por que funções (reuso, nome que documenta a intenção); (c) parâmetros e argumentos; (d) argumento padrão (`def potencia(base, expoente=2)`); (e) `import math` como exemplo de usar um *módulo* pronto; mencionar que `import` traz funções prontas (módulos da biblioteca padrão). Incluir um `.callout` sobre variáveis locais (o que é criado dentro da função fica na função).
- **Na engenharia (3 cards):**
  - ⚙️ Mecânica: `def tensao_normal(forca, area): return forca / area` (tensão = F/A), com uma chamada e `print`.

### Task 3: Módulo 10 — Recursividade

**Interfaces:**

- Produces: fragmento da lição 10. Id de rota: `recursividade`.

**Design do módulo:**
- **Analogia (🪆/📡):** recursão é como *bonecas russas (matrioscas)* — cada uma contém uma menor, até a menor de todas (o caso-base). Em telecom, antenas fractais repetem a mesma forma em escalas menores; resolver um problema dividindo-o numa versão menor de si mesmo.
- **Ilustração SVG:** uma "pilha" de chamadas para `fatorial(3)` → `fatorial(2)` → `fatorial(1)` (caso-base), com setas mostrando a volta multiplicando; caixas aninhadas/empilhadas.
- **Explicação progressiva:** (a) uma função pode chamar a si mesma; (b) precisa de um **caso-base** (quando parar) e de um **passo recursivo** (chama uma versão menor); exemplo `contagem_regressiva(n)`; (c) `fatorial` recursivo; (d) comparar mentalmente com a versão em laço. `.callout`: sem caso-base, a recursão nunca termina (estouro de pilha) — análogo ao laço infinito.
- **Na engenharia (3 cards):**
  - ⚙️ Mecânica: `def fatorial(n): return 1 if n <= 1 else n * fatorial(n-1)` (combinações em análise).
  - ⚡ Elétrica: `def soma_serie(n): return 0 if n == 0 else 1/(2**n) + soma_serie(n-1)` (soma de uma série geométrica — atenuação por estágios).
  - 📡 Telecom: `def fib(n): return n if n < 2 else fib(n-1) + fib(n-2)` (crescimento recursivo).
