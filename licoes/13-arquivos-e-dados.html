Arquivos e dados

⚙️

Pense num caderno de laboratório: você anota cada medição numa linha, guarda o caderno, e depois pode consultá-lo quando quiser. Um arquivo é exatamente isso: um caderno de dados persistente. O formato CSV (comma-separated values, valores separados por vírgula) é a planilha mais simples do mundo: cada linha tem os campos separados por vírgulas, como tempo,tensao / 0,0.0 / 1,1.8. É o formato padrão de troca de medições em laboratórios de engenharia.

Nota importante: aqui no navegador o arquivo é gravado numa memória temporária: ele some ao recarregar a página. Por isso, cada exemplo abaixo primeiro grava o arquivo e depois lê o mesmo arquivo, tudo no mesmo bloco. Na prática, em Python real no seu computador, o arquivo fica salvo em disco e você pode abri-lo novamente depois.

Gravando e lendo um arquivo

Use with open("nome.txt", "w") as f: para abrir (e criar) um arquivo no modo de escrita. Depois use with open("nome.txt") as f: (modo de leitura, padrão) para ler o conteúdo. O with garante que o arquivo é fechado automaticamente ao final do bloco.

Python 
# Grava o arquivo
with open("dados.txt", "w") as f:
    f.write("Olá, arquivo!\n")
    f.write("Segunda linha.\n")

# Lê e imprime o conteúdo
with open("dados.txt") as f:
    print(f.read())

Escrevendo várias linhas com um laço

Um padrão muito comum é gravar um conjunto de medições num laço e depois ler todas de uma vez:

Python 
with open("numeros.txt", "w") as f:
    for n in range(1, 6):
        f.write(f"{n}\n")

with open("numeros.txt") as f:
    print(f.read())

Lendo linha por linha

Para processar cada linha individualmente, percorra o arquivo com for linha in f. Use .strip() para remover o \n do final de cada linha:

Python 
with open("numeros.txt", "w") as f:
    f.write("10\n20\n30\n")

with open("numeros.txt") as f:
    for linha in f:
        print(linha.strip())

CSV: escrevendo e lendo com split(",")

Para um CSV, escreva o cabeçalho na primeira linha e depois os dados. Na leitura, pule o cabeçalho com next(f) e separe os campos com .split(","):

Python 
# Grava CSV
with open("medicoes.csv", "w") as f:
    f.write("tempo,tensao\n")
    f.write("0,0.0\n")
    f.write("1,1.8\n")
    f.write("2,3.0\n")

# Lê e processa o CSV
with open("medicoes.csv") as f:
    next(f)  # pula o cabeçalho
    for linha in f:
        partes = linha.split(",")
        t = float(partes[0])
        v = float(partes[1])
        print(f"t={t}s  v={v}V")
Sempre use with open(...): ele fecha o arquivo automaticamente, mesmo que ocorra um erro. Abrir sem with e esquecer de fechar pode corromper dados.

Memória temporária no navegador: aqui, cada bloco de código é independente: o arquivo some entre execuções. Por isso, cada bloco grava antes de ler. Em Python real no seu computador, o arquivo fica em disco e persiste entre execuções.

Na engenharia

Arquivos CSV aparecem em todas as engenharias para registrar e analisar medições.

⚙️ Mecânica

Gravar 3 medições de força num arquivo e ler de volta para conferência.

Python 
with open("forcas.txt", "w") as f:
    f.write("100.5 N\n")
    f.write("102.3 N\n")
    f.write("98.7 N\n")

with open("forcas.txt") as f:
    for linha in f:
        print(linha.strip())
⚡ Elétrica

Gravar um CSV tempo,tensao e somar a coluna de tensão.

Python 
with open("tensoes.csv", "w") as f:
    f.write("tempo,tensao\n")
    f.write("0,1.2\n")
    f.write("1,3.5\n")
    f.write("2,4.8\n")

soma = 0.0
with open("tensoes.csv") as f:
    next(f)
    for linha in f:
        soma += float(linha.split(",")[1])
print(f"Soma das tensões: {soma:.1f} V")
📡 Telecomunicações

Gravar amostras de sinal e contar quantas linhas o arquivo tem.

Python 
import math
with open("amostras.txt", "w") as f:
    for n in range(8):
        s = math.sin(2 * math.pi * n / 8)
        f.write(f"{s:.4f}\n")

with open("amostras.txt") as f:
    linhas = f.readlines()
print(f"Total de amostras: {len(linhas)}")

12. (⚡ Elétrica) Grave um CSV tempo,tensao para t = 0..5 com tensão de carga de capacitor v = 5 × (1 − e−t), leia os dados e imprima. Depois plote tensão × tempo lida do arquivo.

Mostrar solução
Sem gráfico (texto)
Python 
import math

with open("rc.csv", "w") as f:
    f.write("tempo,tensao\n")
    for n in range(6):
        t = n * 1.0
        v = 5 * (1 - math.exp(-t))
        f.write(f"{t},{v:.3f}\n")

with open("rc.csv") as f:
    print(f.read())
Com gráfico (matplotlib)
Python 
import math
import matplotlib.pyplot as plt

with open("rc.csv", "w") as f:
    f.write("tempo,tensao\n")
    for n in range(6):
        t = n * 1.0
        v = 5 * (1 - math.exp(-t))
        f.write(f"{t},{v:.3f}\n")

ts = []
vs = []
with open("rc.csv") as f:
    next(f)
    for linha in f:
        a, b = linha.split(",")
        ts.append(float(a))
        vs.append(float(b))
plt.plot(ts, vs, marker="o")
plt.title("Tensão lida do CSV")
plt.xlabel("t (s)")
plt.ylabel("V")
plt.grid(True)
plt.show()

15. (Geral) Grave as medições [10, 14, 9, 16, 12, 18, 11] num arquivo, leia e calcule a média. Depois plote os valores com uma linha tracejada horizontal indicando a média.

Mostrar solução
Sem gráfico (texto)
Python 
valores = [10, 14, 9, 16, 12, 18, 11]
with open("med.csv", "w") as f:
    for v in valores:
        f.write(f"{v}\n")

lidos = []
with open("med.csv") as f:
    for linha in f:
        lidos.append(int(linha.strip()))
print("média:", sum(lidos) / len(lidos))
Com gráfico (matplotlib)
Python 
import matplotlib.pyplot as plt

valores = [10, 14, 9, 16, 12, 18, 11]
with open("med.csv", "w") as f:
    for v in valores:
        f.write(f"{v}\n")
