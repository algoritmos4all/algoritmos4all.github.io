Strings

📡

Uma string é uma sequência de caracteres, como uma mensagem de texto transmitida, caractere por caractere, cada um ocupando uma posição numerada. Manipular texto é fatiar, juntar e transformar essa sequência: contar os bits de uma mensagem, padronizar um código de peça, extrair valores de um protocolo separado por vírgula.

Criando e medindo uma string

Em Python, strings são escritas entre aspas (simples ou duplas). A função len() conta quantos caracteres ela tem:

Python 
s = "Engenharia"
print(len(s))   # 10 caracteres
print(s[0])     # primeiro: E
print(s[-1])    # último: a

Fatiamento (slicing)

Assim como listas, strings podem ser fatiadas com [início:fim]. O caractere do índice fim não é incluído:

Python 
s = "Engenharia"
print(s[0:3])   # Eng
print(s[3:])    # enharia  (até o final)
print(s[::-1])  # airanhegnE  (invertida)

Métodos de transformação

Strings possuem métodos que retornam versões modificadas. Os mais usados:

Python 
s = "  Engenharia  "
print(s.upper())            # maiúsculas
print(s.lower())            # minúsculas
print(s.strip())            # remove espaços das pontas
print(s.strip().replace("a", "@"))  # substitui letras
Strings são imutáveis: os métodos não alteram a string original: eles sempre retornam uma nova string. Para guardar o resultado, atribua a uma variável: nova = s.strip().upper().

Dividindo com split()

O método .split(separador) divide a string em partes e retorna uma lista. Muito usado para extrair valores de protocolos e arquivos de dados:

Python 
frase = "sinal de radio frequente"
partes = frase.split(" ")
print(partes)
print(len(partes))  # 4 palavras

Percorrendo caractere por caractere

O laço for letra in s visita cada caractere em ordem, sem precisar de índice:

Python 
s = "SINAL"
for letra in s:
    print(letra)

Na engenharia

Texto aparece em toda interface de engenharia: mensagens de protocolo, códigos de peça, dados de sensores.

📡 Telecomunicações

Contar caracteres de uma mensagem transmitida:

Python 
msg = "ola mundo"
print(len(msg))  # 9 caracteres
⚙️ Mecânica

Padronizar um código de peça (tirar espaços, converter para maiúsculas):

Python 
codigo = "  pe-204 "
print(codigo.strip().upper())  # PE-204
⚡ Elétrica

Extrair amostras de "12;5;9" e somar como inteiros:

Python 
dados = "12;5;9"
partes = dados.split(";")
soma = 0
for p in partes:
    soma += int(p)
print(soma)  # 26

12. (📡 Telecom) Calcule a frequência de cada letra de "telecomunicacoes" usando um dicionário. Imprima as contagens e depois plote um gráfico de barras.

Mostrar solução
Sem gráfico (texto)
Python 
msg = "telecomunicacoes"
freq = {}
for letra in msg:
    freq[letra] = freq.get(letra, 0) + 1
for letra in sorted(freq):
    print(letra, freq[letra])
Com gráfico (matplotlib)
Python 
import matplotlib.pyplot as plt
msg = "telecomunicacoes"
freq = {}
for letra in msg:
    freq[letra] = freq.get(letra, 0) + 1
letras = sorted(freq)
valores = [freq[l] for l in letras]
plt.bar(letras, valores)
plt.title("Frequência de letras")
plt.xlabel("letra")
plt.ylabel("ocorrências")
plt.show()

15. (Geral) Conte quantas palavras de cada comprimento há na frase "o rato roeu a roupa do rei de roma". Imprima a contagem e depois plote um gráfico de barras.

Mostrar solução
Sem gráfico (texto)
Python 
frase = "o rato roeu a roupa do rei de roma"
contagem = {}
for palavra in frase.split():
    n = len(palavra)
    contagem[n] = contagem.get(n, 0) + 1
for n in sorted(contagem):
    print(f"{n} letras: {contagem[n]} palavra(s)")
Com gráfico (matplotlib)
Python 
import matplotlib.pyplot as plt
frase = "o rato roeu a roupa do rei de roma"
contagem = {}
for palavra in frase.split():
    n = len(palavra)
    contagem[n] = contagem.get(n, 0) + 1
tamanhos = sorted(contagem)
qtd = [contagem[n] for n in tamanhos]
plt.bar(tamanhos, qtd)
plt.title("Palavras por comprimento")
plt.xlabel("nº de letras")
plt.ylabel("nº de palavras")
plt.show()
