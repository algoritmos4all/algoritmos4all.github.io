Condicionais

Até agora, nossos programas executavam todas as linhas, sempre na mesma ordem. Mas a parte mais poderosa da programação é deixar o programa tomar decisões: fazer uma coisa em um caso e outra coisa em outro. Para isso usamos o if (que significa "se").

⚙️

Uma condicional é uma decisão automática de controle: como um sistema que para a máquina se a temperatura passar do limite, ou um relé que desliga se a corrente for alta demais. O if escolhe a ação conforme a leitura do sensor — sem precisar de um operador humano a cada instante.

O if

A ideia é simples: "se tal condição for verdadeira, então execute estes passos".

Python 
idade = 20

if idade >= 18:
    print("Você é maior de idade.")
    print("Pode tirar a carteira de motorista.")
Dois detalhes essenciais do Python: a linha do if termina com dois-pontos (:), e o que vem "dentro" do if precisa ser indentado — recuado com espaços (o padrão é 4). A indentação é o que diz ao Python quais linhas pertencem ao bloco.

O else: o "senão"

O else define o que fazer quando a condição do if não é verdadeira.

Python 
idade = 15

if idade >= 18:
    print("Maior de idade.")
else:
    print("Menor de idade.")

O elif: vários casos

Quando há mais de duas possibilidades, usamos elif (junção de "else if", ou "senão, se"). O Python testa as condições de cima para baixo e para na primeira que for verdadeira.

Python 
nota = 7.5

if nota >= 9:
    print("Conceito A")
elif nota >= 7:
    print("Conceito B")
elif nota >= 5:
    print("Conceito C")
else:
    print("Reprovado")

Juntando com input()

Agora dá para fazer programas que reagem ao que a pessoa digita. Execute e responda à pergunta:

Python 
senha = input("Digite a senha: ")

if senha == "python123":
    print("Acesso liberado! ✅")
else:
    print("Senha incorreta. ❌")

Na engenharia

O mesmo if/elif/else resolve decisões das três engenharias. Repare no padrão.

⚙️ Mecânica

Classificar uma medida quanto à tolerância de fabricação:

Python 
medida = 10.3

if medida > 10.2:
    print("Acima da tolerância")
else:
    print("OK")
⚡ Elétrica

Decidir a ação de proteção conforme o nível de tensão:

Python 
tensao = 245

if tensao > 250:
    print("Desligar!")
elif tensao > 230:
    print("Alerta")
else:
    print("Normal")
📡 Telecomunicações

Avaliar a qualidade do sinal recebido (em dBm):

Python 
sinal = -80

if sinal > -60:
    print("Excelente")
else:
    print("Fraco")

Prévia: aonde isso leva

Você ainda não precisa entender este código — execute e veja como condicionais viram um gráfico de barras com linhas-limite de tensão, algo que você vai explorar mais adiante.

Python 
import matplotlib.pyplot as plt
tensoes = [205, 218, 232, 248, 255]
plt.bar([1, 2, 3, 4, 5], tensoes)
plt.axhline(230, color="orange", linestyle="--")
plt.axhline(250, color="red", linestyle="--")
plt.title("Leituras de tensão e limites (prévia)")
plt.xlabel("leitura")
plt.ylabel("tensão (V)")
plt.show()
