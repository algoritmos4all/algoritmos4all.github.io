Vetores e listas

📡

Pense num registrador de dados (data logger) acoplado a um sensor: a cada instante ele grava uma nova medição — temperatura, tensão, aceleração — num slot numerado. No final você tem uma série de medições em ordem, cada uma acessível pelo seu número de posição. Em Python essa estrutura se chama lista (e em outras linguagens costuma ser chamada de vetor ou array). Guardar, percorrer e processar essa sequência de amostras é a operação mais comum em programação de engenharia.

Criando uma lista

Uma lista é escrita entre colchetes, com os elementos separados por vírgula. Pode conter números, textos ou qualquer outro tipo de valor:

Python 
medidas = [12, 7, 9, 4, 5]
print(medidas)
print(type(medidas))

Acessando elementos pelo índice

Use colchetes com o número do índice para ler um elemento. O primeiro é [0]; o último pode ser acessado com [-1] (Python conta de trás para frente com índices negativos):

Python 
medidas = [12, 7, 9, 4, 5]
print(medidas[0])   # primeiro: 12
print(medidas[-1])  # último: 5

Tamanho com len()

A função embutida len() retorna quantos elementos a lista tem:

Python 
medidas = [12, 7, 9, 4, 5]
print(len(medidas))  # 5
Índice começa em 0: se uma lista tem 5 elementos, os índices válidos são 0, 1, 2, 3 e 4. Tentar acessar lista[5] causa um IndexError — erro de índice fora do intervalo. Sempre verifique os limites!

Percorrendo com for

O laço for x in lista passa por cada elemento automaticamente, sem precisar gerenciar índice:

Python 
medidas = [12, 7, 9, 4, 5]
for x in medidas:
    print(x)

Construindo uma lista com append

Começamos com uma lista vazia [] e usamos .append(x) para adicionar um elemento de cada vez. Esse padrão é muito usado para construir listas a partir de cálculos:

Python 
lista = []
for n in range(1, 6):
    lista.append(n * 2)
print(lista)  # [2, 4, 6, 8, 10]

Somando com acumulador

O padrão acumulador que aprendemos nos laços funciona perfeitamente com listas. Aqui somamos todas as medições:

Python 
medidas = [12, 7, 9, 4, 5]
soma = 0
for x in medidas:
    soma += x
print(f"Soma = {soma}")
print(f"Média = {soma / len(medidas)}")

Na engenharia

Listas aparecem em toda medição e simulação de engenharia. Veja o padrão nas três áreas.

⚙️ Mecânica

Lista de posições registradas; imprime cada posição com o instante de tempo (índice):

Python 
pos = [0.0, 1.2, 4.9, 11.0]
for i in range(len(pos)):
    print(f"t={i}  pos={pos[i]} m")
⚡ Elétrica

Lista de tensões medidas; calcula e imprime a média:

Python 
tensoes = [3.1, 3.3, 3.0, 3.2, 3.4]
s = 0
for v in tensoes:
    s += v
media = s / len(tensoes)
print(f"Média = {media:.2f} V")
📡 Telecomunicações

Constrói lista de amostras e conta quantas estão acima de zero:

Python 
import math
amostras = []
for n in range(10):
    t = n * 0.1
    amostras.append(math.sin(2 * math.pi * t))
positivas = 0
for a in amostras:
    if a > 0:
        positivas += 1
print(f"{positivas} amostras acima de 0")

11. Dada temps = [18, 25, 20, 30, 22], calcule a média e depois conte quantas temperaturas estão acima dela. Imprima a média e o contador.

Mostrar solução
Python 
temps = [18, 25, 20, 30, 22]
media = sum(temps) / len(temps)  # sum() soma a lista de uma vez (atalho do acumulador)
c = 0
for t in temps:
    if t > media:
        c += 1
print(f"média={media}, acima={c}")

13. (⚙️ Mecânica) Construa listas de instantes de tempo e posições de queda livre (y = 0,5 × 9,81 × t²) para t = n × 0,2 s, com n de 0 a 15. Imprima os pares t, y. Depois plote a trajetória.

Mostrar solução
Sem gráfico (texto)
Python 
g = 9.81
ts = []
ys = []
for n in range(16):
    t = n * 0.2
    ts.append(t)
    ys.append(0.5 * g * t**2)
for i in range(len(ts)):
    print(f"t={ts[i]:.1f}s  y={ys[i]:.2f} m")
Com gráfico (matplotlib)
Python 
import matplotlib.pyplot as plt
g = 9.81
ts = [n * 0.2 for n in range(16)]
ys = [0.5 * g * t**2 for t in ts]
plt.plot(ts, ys, marker=".")
plt.title("Queda livre")
plt.xlabel("t (s)")
plt.ylabel("y (m)")
plt.grid(True)
plt.show()

14. (📡 Telecom) Construa uma lista de 51 amostras de uma senoide de 3 Hz (t = n × 0,02 s, n de 0 a 50). Imprima quantas amostras foram geradas e os valores da primeira e da última. Depois plote a forma de onda.

Mostrar solução
Sem gráfico (texto)
Python 
import math
amostras = []
for n in range(51):
    t = n * 0.02
    amostras.append(math.sin(2 * math.pi * 3 * t))
print(f"{len(amostras)} amostras geradas")
print(amostras[0], amostras[-1])
Com gráfico (matplotlib)
Python 
import math
import matplotlib.pyplot as plt
ts = [n * 0.02 for n in range(51)]
amostras = [math.sin(2 * math.pi * 3 * t) for t in ts]
plt.plot(ts, amostras)
plt.title("Senoide 3 Hz")
plt.xlabel("t (s)")
plt.ylabel("amplitude")
plt.grid(True)
plt.show()

15. (Geral) Dada a lista valores = [2, 8, 4, 10, 6, 12, 8], construa a lista de médias móveis de pares de vizinhos ((valores[i] + valores[i+1]) / 2). Imprima as duas listas. Depois plote original vs suavizada.

Mostrar solução
Sem gráfico (texto)
Python 
valores = [2, 8, 4, 10, 6, 12, 8]
suave = []
for i in range(len(valores) - 1):
    suave.append((valores[i] + valores[i+1]) / 2)
print("original:", valores)
print("suavizada:", suave)
Com gráfico (matplotlib)
Python 
import matplotlib.pyplot as plt
valores = [2, 8, 4, 10, 6, 12, 8]
suave = [(valores[i] + valores[i+1]) / 2 for i in range(len(valores) - 1)]
plt.plot(range(len(valores)), valores, marker="o", label="original")
plt.plot(range(len(suave)), suave, marker="s", label="média móvel")
plt.title("Suavização")
plt.xlabel("índice")
plt.ylabel("valor")
plt.legend()
plt.grid(True)
plt.show()
