O que é um algoritmo

Antes de escrever qualquer código, vale entender a ideia mais importante de toda a programação: o algoritmo.

Um algoritmo é simplesmente uma sequência de passos para resolver um problema ou realizar uma tarefa. Você já segue algoritmos o dia todo, mesmo sem perceber. Uma receita de bolo é um algoritmo. As instruções para chegar à casa de um amigo são um algoritmo.

Em uma frase: programar é descrever, com muita clareza e na ordem certa, os passos que o computador deve seguir.

Um exemplo do dia a dia

Pense no algoritmo para "fazer um sanduíche":

  1. Pegar duas fatias de pão.
  2. Passar manteiga em uma fatia.
  3. Colocar o queijo sobre a manteiga.
  4. Cobrir com a outra fatia.

Repare em duas coisas: a ordem importa (não dá para cobrir antes de colocar o queijo) e os passos precisam ser precisos. O computador é muito obediente, mas nada esperto: ele faz exatamente o que você manda, na ordem em que você manda.

O primeiro algoritmo em Python

Em Python, o comando print() mostra uma mensagem na tela. Este é o tradicional "primeiro programa" de quem aprende a programar. Clique em Executar para vê-lo funcionar:

Python 
print("Olá, mundo!")
print("Estou aprendendo a programar.")

Cada linha é um passo. O computador executa de cima para baixo: primeiro mostra a primeira mensagem, depois a segunda. Esse é o nosso algoritmo em ação.

Por que a ordem importa

Experimente trocar a ordem das linhas abaixo (você pode editar o código antes de executar) e veja como a saída muda:

Python — edite à vontade 
print("Bom dia!")
print("Como vai você?")
print("Até logo!")
Dica: o código aqui é editável. Errar não tem problema nenhum — pelo contrário, é assim que se aprende. Mude, execute, observe.

Exercícios

1. Escreva um programa que mostre seu nome e sua cidade, cada um em uma linha.

Mostrar solução
Python 
print("Maria Silva")
print("Bagé")

2. Descreva, em passos numerados (em português mesmo, sem código), o algoritmo para "escovar os dentes". Pense em pelo menos quatro passos na ordem certa.

Mostrar uma solução possível
  1. Pegar a escova e a pasta de dente.
  2. Colocar pasta sobre a escova.
  3. Escovar os dentes por dois minutos.
  4. Enxaguar a boca e a escova.

Não existe uma única resposta certa — o importante é que os passos estejam claros e na ordem correta.
