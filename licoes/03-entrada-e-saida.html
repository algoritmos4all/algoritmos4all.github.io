Entrada e saída

📡

Pense num instrumento de medição com mostrador: ele recebe uma leitura do mundo externo (entrada) e exibe o resultado no display (saída). Um programa faz o mesmo: lê dados da pessoa que o usa e devolve uma resposta. input() é o sensor; print() é o display.

Programas ficam muito mais interessantes quando conversam com a pessoa que os usa. Para isso existem duas operações fundamentais:

  • Saída: mostrar informação — já usamos print().
  • Entrada: receber informação — feito com input().

Mostrando informação com print()

O print() aceita vários valores separados por vírgula e os exibe com um espaço entre eles:

Python 
print("Resultado:", 7, "pontos")

Recebendo informação com input()

O input() faz uma pergunta e espera a pessoa digitar uma resposta. O que for digitado vira o valor de uma variável.

Python 
nome = input("Qual é o seu nome? ")
print(f"Prazer em conhecer você, {nome}!")
Como funciona aqui no site: ao executar, o navegador abre uma janelinha pedindo o valor. Digite algo e confirme para o programa continuar.

Tudo que vem do input() é texto

Um detalhe que pega muita gente: o input() sempre devolve texto (str), mesmo que a pessoa digite números. Veja o que acontece ao tentar somar duas "idades":

Python 
a = input("Primeiro número: ")
b = input("Segundo número: ")
print(a + b)

Se você digitar 2 e 3, o resultado será 23 — porque o + entre textos os junta, em vez de somar.

Convertendo texto em número

Para fazer contas, convertemos o texto para número com int() (inteiro) ou float() (decimal):

Python 
a = int(input("Primeiro número: "))
b = int(input("Segundo número: "))
print("A soma é:", a + b)
Regra de bolso: recebeu com input() e vai fazer conta? Converta antes com int() ou float().

Na engenharia

Em engenharia, quase todo cálculo começa com a leitura de uma grandeza medida e termina com a exibição de um resultado. Veja o padrão: input() → conversão → aritmética → print().

⚙️ Mecânica

Ler força (N) e área (m²) e calcular a tensão normal F/A (Pa):

Python 
f = float(input("Força (N): "))
a = float(input("Área (m²): "))
print(f"Tensão = {f / a} Pa")
⚡ Elétrica

Ler tensão (V) e resistência (Ω) e calcular a corrente V/R pela Lei de Ohm:

Python 
v = float(input("Tensão (V): "))
r = float(input("Resistência (Ω): "))
print(f"Corrente = {v / r} A")
📡 Telecomunicações

Ler a frequência (Hz) e calcular o período T = 1/f (s):

Python 
f = float(input("Frequência (Hz): "))
print(f"Período = {1 / f} s")

Faça na ordem. 🟢 aquecem a sintaxe, 🟡 combinam ideias, 🔴 aplicam à engenharia. Tente antes de abrir a solução.

1. Pergunte a cidade da pessoa e responda com a frase "Que legal, eu também adoro <cidade>!".

Mostrar solução
Python

Python

print(f"{nome}, {idade} anos")
🟢 Aquecimento

10. (⚙️ Mecânica) Leia a massa de um corpo (kg) e mostre o seu peso na Terra, usando g = 9,81 m/s² (peso = m × g).

Mostrar solução
Python 
m = float(input("Massa (kg): "))
print(f"Peso = {m * 9.81} N")
🔴 Desafio de engenharia

11. (⚡ Elétrica) Leia a tensão (V) e a corrente (A) de um circuito e mostre a potência elétrica (P = V × I).

Mostrar solução
Python

print(f"Potência = {v * i} W")

12. (📡 Telecom) A velocidade da luz é c = 3×10⁸ m/s. Leia a frequência de uma onda eletromagnética (Hz) e calcule o comprimento de onda λ = c / f.

Mostrar solução
Python

f = float(input("Frequência (Hz): "))
print(f"λ = {c / f} m")

Prévia: aonde isso leva

Você ainda não precisa entender este código — execute e veja os dados lidos com input() sendo usados para montar um gráfico, algo que você vai aprender a fazer mais adiante.

Python 
import matplotlib.pyplot as plt
plt.plot([0, 1, 2, 3, 4], [0, 9.81, 19.62, 29.43, 39.24], marker="o")
plt.title("Peso vs. massa (prévia)")
plt.xlabel("massa (kg)")
plt.ylabel("peso (N)")
plt.grid(True)
plt.show()
