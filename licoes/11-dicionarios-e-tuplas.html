Dicionários e tuplas

⚡

Um dicionário é uma tabela de consulta: você fornece uma chave (o nome de um material, o código de cor de um resistor, o número de um canal Wi-Fi) e obtém imediatamente o valor associado, sem precisar percorrer tudo, posição por posição. É diferente de uma lista, onde você precisa saber o índice numérico. Uma tupla é um par fixo (ou trio, ou quartet…) que não muda depois de criado, como a coordenada (x, y) de um ponto, ou um par tensão/corrente medido no laboratório.

Criando um dicionário e acessando por chave

Um dicionário é escrito com chaves {}. Cada entrada é um par chave: valor separado por vírgula. Para acessar um valor, use d["chave"]:

Python 
aluno = {"nome": "Ana", "idade": 20}
print(aluno["nome"])
print(aluno["idade"])

Adicionando e atualizando entradas

Para adicionar uma nova chave basta atribuir um valor a ela. Atribuir a uma chave já existente atualiza o valor:

Python 
aluno = {"nome": "Ana", "idade": 20}
aluno["curso"] = "Eng"        # nova chave
aluno["idade"] = 21           # atualiza existente
print(aluno)

Percorrendo um dicionário

for chave in d percorre as chaves. Para obter chave e valor ao mesmo tempo, use d.items():

Python 
notas = {"mat": 8, "fis": 9, "qui": 7}

# percorrer só as chaves
for chave in notas:
    print(chave)

# percorrer chave e valor juntos
for chave, valor in notas.items():
    print(chave, "->", valor)

Tamanho com len()

len(d) retorna o número de pares chave-valor do dicionário:

Python 
notas = {"mat": 8, "fis": 9, "qui": 7}
print(len(notas))  # 3

Tuplas e desempacotamento

Uma tupla é escrita com parênteses. Você pode "desempacotar" seus elementos em variáveis separadas numa só linha, ótimo para coordenadas e pares de medidas:

Python 
ponto = (3, 4)
x, y = ponto        # desempacotamento
print(f"x = {x}, y = {y}")
KeyError: acessar d["chave_inexistente"] causa um KeyError. Use d.get("chave", valor_padrão) para obter um valor de segurança em vez de um erro.

Tupla é imutável: diferente de uma lista, você não pode alterar um elemento de uma tupla depois de criá-la: qualquer tentativa gera TypeError.

Na engenharia

Dicionários e tuplas aparecem em tabelas de propriedades e pares de medidas nas três engenharias.

⚡ Elétrica

Código de cores de resistores: a cor aponta direto para o dígito.

Python 
cores = {"preto": 0, "marrom": 1, "vermelho": 2}
print(cores["vermelho"])
⚙️ Mecânica

Tabela de densidades de materiais (kg/m³) consultada pelo nome.

Python 
densidades = {"aço": 7850, "alumínio": 2700, "cobre": 8960}
print(densidades["alumínio"])
📡 Telecomunicações

Canais Wi-Fi 2,4 GHz: canal → frequência (MHz).

Python 
canais = {1: 2412, 6: 2437, 11: 2462}
print(canais[6])

10. Use um dicionário contador para contar quantos números de 1 a 10 são pares e quantos são ímpares. Imprima o dicionário. (Esperado: {'par': 5, 'impar': 5}.)

Mostrar solução
Python 
contagem = {"par": 0, "impar": 0}
for n in range(1, 11):
    if n % 2 == 0:
        contagem["par"] += 1
    else:
        contagem["impar"] += 1
print(contagem)

12. (⚡ Elétrica) Dado medicoes = {0: 0.0, 1: 1.8, 2: 3.0, 3: 3.7, 4: 4.1} (tempo em s → tensão em V), imprima cada par no texto e plote tensão × tempo.

Mostrar solução
Sem gráfico (texto)
Python 
medicoes = {0: 0.0, 1: 1.8, 2: 3.0, 3: 3.7, 4: 4.1}
for t, v in medicoes.items():
    print(f"t={t}s  v={v} V")
Com gráfico (matplotlib)
Python 
import matplotlib.pyplot as plt
medicoes = {0: 0.0, 1: 1.8, 2: 3.0, 3: 3.7, 4: 4.1}
tempos = list(medicoes.keys())
tensoes = list(medicoes.values())
plt.plot(tempos, tensoes, marker="o")
plt.title("Medições")
plt.xlabel("t (s)")
plt.ylabel("V")
plt.grid(True)
plt.show()

15. (Geral) Construa um histograma contando quantas vezes cada dígito das unidades aparece nos números de 1 a 20. Imprima o resultado e plote em barras.

Mostrar solução
Sem gráfico (texto)
Python 
contagem = {}
for n in range(1, 21):
    d = n % 10
    contagem[d] = contagem.get(d, 0) + 1
for d in sorted(contagem):
    print(d, contagem[d])
Com gráfico (matplotlib)
Python 
import matplotlib.pyplot as plt
contagem = {}
for n in range(1, 21):
    d = n % 10
    contagem[d] = contagem.get(d, 0) + 1
digitos = sorted(contagem)
freq = [contagem[d] for d in digitos]
plt.bar(digitos, freq)
plt.title("Frequência do dígito das unidades (1–20)")
plt.xlabel("dígito")
plt.ylabel("ocorrências")
plt.show()
