Juntando tudo

Parabéns por chegar até aqui! 🎉 Você já conhece os blocos fundamentais da programação: variáveis, entrada e saída, operadores, condicionais e laços. Nesta última lição vamos combinar tudo em pequenos projetos — porque é juntando as peças que a programação ganha vida.

⚙️

"Juntando tudo" é como montar um instrumento completo a partir das peças: variáveis guardam as leituras, condicionais tomam decisões e laços repetem o processo — juntos formam uma rotina de medição e controle de verdade, como um sistema de monitoramento que amostra um sensor continuamente, compara cada leitura com um limite e reage automaticamente.

Projeto 1 — Adivinhe o número

Um jogo clássico: o programa "pensa" em um número e a pessoa tenta adivinhar, recebendo dicas de "maior" ou "menor". Ele usa laço, condicional e entrada ao mesmo tempo.

Python 
import random

segredo = random.randint(1, 20)
tentativas = 0

print("Pensei em um número de 1 a 20. Tente adivinhar!")

while True:
    palpite = int(input("Seu palpite: "))
    tentativas = tentativas + 1

    if palpite < segredo:
        print("É maior. 🔼")
    elif palpite > segredo:
        print("É menor. 🔽")
    else:
        print(f"Acertou em {tentativas} tentativas! 🎉")
        break
Duas novidades: import random traz ferramentas para sortear números, e break interrompe o laço imediatamente — útil quando a pessoa acerta.

Projeto 2 — Calculadora de média com situação

Junta entrada convertida, conta e decisão para informar o resultado de forma amigável.

Python 
nome = input("Nome do aluno: ")
n1 = float(input("Primeira nota: "))
n2 = float(input("Segunda nota: "))

media = (n1 + n2) / 2
print(f"{nome}, sua média é {media:.1f}")

if media >= 7:
    print("Situação: Aprovado ✅")
elif media >= 5:
    print("Situação: Recuperação ⚠️")
else:
    print("Situação: Reprovado ❌")

O {media:.1f} dentro da f-string mostra o número com apenas uma casa decimal — um detalhe que deixa a saída mais limpa.

Projeto 3 — Soma de uma lista de compras

Usa um laço com acumulador e um contador para somar gastos até a pessoa digitar 0.

Python 
total = 0
itens = 0

print("Digite o preço de cada item (0 para finalizar).")

while True:
    preco = float(input("Preço do item: "))
    if preco == 0:
        break
    total = total + preco
    itens = itens + 1

print(f"Você comprou {itens} itens.")
print(f"Total da compra: R$ {total:.2f}")

Na engenharia

O mesmo padrão — laço + condicional — aparece nos três cursos de engenharia. Repare como cada programa combina as duas ferramentas.

Simular a queda livre e avisar quando a altura passa de 20 m:

Python 
g = 9.81
t = 0.0

5. Crie um pequeno "caixa eletrônico": pergunte um valor de saque e mostre quantas notas de 100, 50, 20 e 10 são necessárias (assuma valores múltiplos de 10). Dica: use // e %.

Mostrar solução
Python 
valor = int(input("Valor do saque (múltiplo de 10): "))

for nota in [100, 50, 20, 10]:
    quantidade = valor // nota
    valor = valor % nota
    if quantidade > 0:
        print(f"{quantidade} nota(s) de R$ {nota}")

Aqui usamos um for percorrendo uma lista de valores de nota — uma prévia do que você vai aprender a seguir! Listas são detalhadas no módulo 9; aqui ela aparece como um atalho conveniente.

10. (⚙️ Mecânica) Simule a queda livre de 0 a 3 s com passo de 0,2 s: imprima cada altura e avise quando passar de 20 m. Depois plote a trajetória.

Mostrar solução
Sem gráfico (texto)
Python 
g = 9.81
t = 0.0
while t <= 3.0:
    altura = 0.5 * g * t**2
    if altura > 20:
        print(f"t={t:.1f}s  altura={altura:.1f} m (passou de 20 m)")
    else:
        print(f"t={t:.1f}s  altura={altura:.1f} m")
    t = t + 0.2
Com gráfico (matplotlib)
Python 
import matplotlib.pyplot as plt

g = 9.81
ts = []
ys = []
t = 0.0
while t <= 3.0:
    ts.append(t)
    ys.append(0.5 * g * t**2)
    t = t + 0.2

plt.plot(ts, ys, marker="o")
plt.axhline(20, color="red", linestyle="--")
plt.title("Queda livre")
plt.xlabel("t (s)")
plt.ylabel("altura (m)")
plt.grid(True)
plt.show()

Prévia do módulo 9: as variáveis ts e ys são listas criadas com .append() — a única forma usada aqui é para coletar os pontos do gráfico. Você vai entender listas completamente no módulo 9.

12. (📡 Telecom) Amostre uma senoide de 2 Hz de 0 a 1 s com passo 0,05 s: conte quantas amostras são positivas e depois plote a forma de onda.

Mostrar solução
Sem gráfico (texto)
Python 
import math

t = 0.0
positivas = 0
while t <= 1.0:
    s = math.sin(2 * math.pi * 2 * t)
    if s > 0:
        positivas = positivas + 1
    t = t + 0.05
print(f"{positivas} amostras positivas")
Com gráfico (matplotlib)
Python 
import math
import matplotlib.pyplot as plt

ts = []
ss = []
t = 0.0
while t <= 1.0:
    ts.append(t)
    ss.append(math.sin(2 * math.pi * 2 * t))
    t = t + 0.05

plt.plot(ts, ss, marker=".")
plt.axhline(0, color="red", linestyle="--")
plt.title("Sinal amostrado")
plt.xlabel("t (s)")
plt.ylabel("amplitude")
plt.grid(True)
plt.show()
