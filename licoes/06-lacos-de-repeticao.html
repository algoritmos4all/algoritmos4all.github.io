Laços de repetição

Computadores são excelentes em repetir tarefas sem cansar nem reclamar. Em vez de escrever a mesma linha cem vezes, usamos laços (em inglês, loops) para repetir um bloco de código. Python tem dois: while e for.

O while: "enquanto"

O while repete um bloco enquanto uma condição for verdadeira. Ele testa a condição, executa o bloco, testa de novo, e assim por diante.

Python 
contador = 1

while contador <= 5:
    print(f"Contando: {contador}")
    contador = contador + 1

print("Acabou!")

Repare na linha contador = contador + 1: ela aumenta o contador a cada volta. Sem ela, a condição seria sempre verdadeira e o laço nunca terminaria.

Laço infinito: se a condição de um while nunca ficar falsa, o programa "trava" repetindo para sempre. Garanta sempre que algo dentro do laço caminhe em direção ao fim.

O for: repetir um número de vezes

Quando sabemos quantas vezes queremos repetir, o for com range() é mais prático. range(5) gera os números de 0 a 4 (começa no zero e para antes do número indicado).

Python 
for i in range(5):
    print(f"Repetição número {i}")

O range() também aceita início e fim. range(1, 6) vai de 1 a 5:

Python 
for numero in range(1, 6):
    print(f"{numero} x 7 = {numero * 7}")

Somando dentro de um laço

Um padrão muito comum é usar uma variável "acumuladora" que cresce a cada volta. Aqui somamos os números de 1 a 100:

Python 
soma = 0

for n in range(1, 101):
    soma = soma + n

print(f"A soma de 1 a 100 é {soma}")

Exercícios

1. Use um for para mostrar a tabuada do 9 (de 9 x 1 até 9 x 10).

Mostrar solução
Python 
for i in range(1, 11):
    print(f"9 x {i} = {9 * i}")

2. Peça um número à pessoa e use um while (ou for) para fazer uma contagem regressiva desse número até 1, terminando com "Fogo! 🚀".

Mostrar solução
Python 
n = int(input("Começar a contagem de quanto? "))

while n >= 1:
    print(n)
    n = n - 1

print("Fogo! 🚀")
