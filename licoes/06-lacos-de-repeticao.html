Laços de repetição

⚡

Pense num osciloscópio medindo uma tensão: ele repete a mesma medição a cada intervalo de tempo, milhares de vezes por segundo. Um laço é exatamente isso: repetir um bloco de instruções sem cansar. Amostrar um sinal, varrer uma faixa de frequências, integrar passo a passo uma simulação: tudo são laços.

O while: "enquanto"

O while repete um bloco enquanto uma condição for verdadeira. Ele testa a condição, executa o bloco, testa de novo, e assim por diante.

Python 
contador = 1

while contador <= 5:
    print(f"Contando: {contador}")
    contador = contador + 1

print("Acabou!")

Repare na linha contador = contador + 1: ela aumenta o contador a cada volta. Sem ela, a condição seria sempre verdadeira e o laço nunca terminaria.

Laço infinito: se a condição de um while nunca ficar falsa, o programa "trava" repetindo para sempre. Garanta sempre que algo dentro do laço caminhe em direção ao fim.

O for: repetir um número de vezes

Quando sabemos quantas vezes queremos repetir, o for com range() é mais prático. range(5) gera os números de 0 a 4 (começa no zero e para antes do número indicado).

Python 
for i in range(5):
    print(f"Repetição número {i}")

O range() também aceita início e fim. range(1, 6) vai de 1 a 5:

Python 
for numero in range(1, 6):
    print(f"{numero} x 7 = {numero * 7}")

Somando dentro de um laço

Um padrão muito comum é usar uma variável "acumuladora" que cresce a cada volta. Aqui somamos os números de 1 a 100:

Python 
soma = 0

for n in range(1, 101):
    soma = soma + n

print(f"A soma de 1 a 100 é {soma}")

Na engenharia

O mesmo laço resolve problemas das três engenharias. Repare no padrão.

⚙️ Mecânica

Posição de um corpo em queda a cada 0,5 s (sem resistência do ar):

Python 
g = 9.81
t = 0.0
while t <= 2.0:
    altura = 0.5 * g * t**2
    print(f"t={t:.1f}s  queda={altura:.2f} m")
    t = t + 0.5
⚡ Elétrica

Tensão de um capacitor carregando, amostrada a cada constante de tempo:

Python 
import math
V = 5.0
for k in range(6):
    v = V * (1 - math.exp(-k))
    print(f"{k} tau -> {v:.3f} V")
📡 Telecomunicações

Amostrando um sinal senoidal de 2 Hz a cada 0,1 s:

Python 
import math
for n in range(10):
    t = n * 0.1
    amostra = math.sin(2 * math.pi * 2 * t)
    print(f"t={t:.1f}s  amostra={amostra:+.3f}")

Caso real de engenharia: média de leituras de um sensor

Sensores entregam muitas medições. Um laço percorre todas, acumula a soma e calcula a média, sem você precisar escrever uma linha por leitura.

Python 
leituras = [21.5, 22.1, 21.8, 23.0, 22.4, 21.9]

soma = 0
for valor in leituras:
    soma = soma + valor

media = soma / len(leituras)
print(f"{len(leituras)} leituras")
print(f"Temperatura media: {media:.2f} graus C")
O padrão "acumular dentro de um laço" calcula somas, médias, máximos e mínimos de qualquer conjunto de dados.

12. (📡 Telecom) Amostre o sinal s(t) = sin(2π·5·t) (5 Hz) de 0 a 1 s com passo 0,02 s. Mostre as amostras e depois plote a forma de onda.

Mostrar solução
Sem gráfico (texto)
Python 
import math
for n in range(51):  # t = n * passo evita erro de ponto flutuante
    t = n * 0.02
    s = math.sin(2 * math.pi * 5 * t)
    print(f"t={t:.2f}s  s={s:+.3f}")
Com gráfico (matplotlib)
Python 
import math
import matplotlib.pyplot as plt
ts, ss = [], []
for n in range(51):  # t = n * passo evita erro de ponto flutuante
    t = n * 0.02
    ts.append(t)
    ss.append(math.sin(2 * math.pi * 5 * t))
plt.plot(ts, ss, marker=".")
plt.title("Sinal 5 Hz amostrado")
plt.xlabel("t (s)")
plt.ylabel("s(t)")
plt.grid(True)
plt.show()

15. (⚙️ Mecânica/geral) Aproxime a soma da série Σ 1/k² (k de 1 a N) e veja como ela converge para π²/6 ≈ 1,6449. Imprima a soma parcial a cada 10 termos até N = 100 e depois plote a convergência com uma linha horizontal em π²/6.

Mostrar solução
Sem gráfico (texto)
Python 
import math
s = 0.0
for k in range(1, 101):
    s += 1 / k**2
    if k % 10 == 0:
        print(f"k={k:3d}  soma={s:.6f}  pi2/6={math.pi**2/6:.6f}")
Com gráfico (matplotlib)
Python 
import math
import matplotlib.pyplot as plt
ks, somas = [], []
s = 0.0
for k in range(1, 101):
    s += 1 / k**2
    ks.append(k)
    somas.append(s)
plt.plot(ks, somas, label="soma parcial")
plt.axhline(math.pi**2 / 6, color="red", linestyle="--", label="π²/6")
plt.title("Convergência de Σ 1/k²")
plt.xlabel("k")
plt.ylabel("soma")
plt.legend()
plt.grid(True)
plt.show()
