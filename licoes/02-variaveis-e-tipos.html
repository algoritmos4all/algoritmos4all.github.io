Variáveis e tipos de dados

Programas trabalham com informações: um nome, uma idade, um preço, uma resposta. Para guardar essas informações e usá-las depois, usamos variáveis.

O que é uma variável

Pense em uma variável como uma caixa com etiqueta. Você guarda um valor dentro dela e dá um nome à caixa. Depois, sempre que precisar daquele valor, basta chamar pelo nome.

Em Python, criamos uma variável com o sinal de igual (=):

Python 
nome = "Ana"
idade = 25

print(nome)
print(idade)

Lê-se assim: "nome recebe "Ana"". O nome da variável fica à esquerda do =; o valor guardado, à direita.

Atenção: em programação, = não significa "igual a" como na matemática. Significa "guarde este valor nesta variável".

Tipos de dados

Cada valor tem um tipo. Por enquanto, três são suficientes:

  • Texto (str): palavras e frases, sempre entre aspas. Ex.: "Olá", "Rua das Flores, 10".
  • Número inteiro (int): números sem casas decimais. Ex.: 7, -3, 2025.
  • Número decimal (float): números com vírgula — que em Python se escreve com ponto. Ex.: 3.14, 19.90.

A função type() revela o tipo de um valor:

Python 
print(type("Ana"))
print(type(25))
print(type(19.90))

Mostrando texto e variáveis juntos

Para misturar texto fixo com o valor de variáveis, a forma mais prática é a f-string: uma string com a letra f antes das aspas, onde colocamos variáveis entre chaves { }.

Python 
nome = "Ana"
idade = 25

print(f"{nome} tem {idade} anos.")

Variáveis podem mudar

O valor guardado pode ser trocado a qualquer momento — por isso o nome "variável":

Python 
pontos = 0
print(f"Início: {pontos} pontos")

pontos = 10
print(f"Depois de acertar: {pontos} pontos")

Exercícios

1. Crie três variáveis — produto, preco e quantidade — e mostre uma frase como "Comprei 3 cadernos a 12.5 cada".

Mostrar solução
Python 
produto = "cadernos"
preco = 12.5
quantidade = 3

print(f"Comprei {quantidade} {produto} a {preco} cada")

2. Sem executar, tente adivinhar o tipo de cada valor: "42", 42 e 4.2. Depois confira com type().

Mostrar solução

"42" é texto (str), por causa das aspas; 42 é inteiro (int); 4.2 é decimal (float).

Python 
print(type("42"))
print(type(42))
print(type(4.2))
