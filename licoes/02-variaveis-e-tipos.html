Variáveis e tipos de dados

⚡

Pense num sensor de tensão ligado a um painel de controle: ao fazer uma leitura, o técnico anota o valor numa etiqueta — "tensão = 12,5 V" — para usar no cálculo seguinte. Uma variável é exatamente isso: uma caixa rotulada onde o programa guarda uma medição (ou qualquer outro dado) para usar quando precisar. O nome é o rótulo; o valor lido é o que está guardado dentro.

O que é uma variável

Programas trabalham com informações: um nome, uma idade, um preço, uma resposta. Para guardar essas informações e usá-las depois, usamos variáveis.

Pense em uma variável como uma caixa com etiqueta. Você guarda um valor dentro dela e dá um nome à caixa. Depois, sempre que precisar daquele valor, basta chamar pelo nome.

Em Python, criamos uma variável com o sinal de igual (=):

Python 
nome = "Ana"
idade = 25

print(nome)
print(idade)

Lê-se assim: "nome recebe "Ana"". O nome da variável fica à esquerda do =; o valor guardado, à direita.

Atenção: em programação, = não significa "igual a" como na matemática. Significa "guarde este valor nesta variável".

Tipos de dados

Cada valor tem um tipo. Por enquanto, três são suficientes:

  • Texto (str): palavras e frases, sempre entre aspas. Ex.: "Olá", "Rua das Flores, 10".
  • Número inteiro (int): números sem casas decimais. Ex.: 7, -3, 2025.
  • Número decimal (float): números com vírgula — que em Python se escreve com ponto. Ex.: 3.14, 19.90.

A função type() revela o tipo de um valor:

Python 
print(type("Ana"))
print(type(25))
print(type(19.90))

Mostrando texto e variáveis juntos

Para misturar texto fixo com o valor de variáveis, a forma mais prática é a f-string: uma string com a letra f antes das aspas, onde colocamos variáveis entre chaves { }.

Python 
nome = "Ana"
idade = 25

print(f"{nome} tem {idade} anos.")

Variáveis podem mudar

O valor guardado pode ser trocado a qualquer momento — por isso o nome "variável":

Python 
pontos = 0
print(f"Início: {pontos} pontos")

pontos = 10
print(f"Depois de acertar: {pontos} pontos")

Na engenharia

Guardar grandezas físicas em variáveis e combiná-las com aritmética é a base de qualquer cálculo de engenharia em Python. Veja três exemplos.

⚙️ Mecânica

Segunda Lei de Newton: força = massa × aceleração (F = m·a).

Python 
massa = 1200
aceleracao = 3.5
print(f"Força = {massa * aceleracao} N")
⚡ Elétrica

Potência elétrica: P = V × I (tensão vezes corrente).

Python 
tensao = 12.0
corrente = 2.0
print(f"Potência = {tensao * corrente} W")
📡 Telecomunicações

Frequência de operação de um rádio Wi-Fi guardada numa variável.

Python 
frequencia = 2.4
print(f"Frequência: {frequencia} GHz")

Mostrar solução
Python

preco = 12.5
quantidade = 3

print(f"Comprei {quantidade} {produto} a {preco} cada")
🟢 Aquecimento

2. Crie as variáveis cidade e estado e imprima a frase "Moro em X, Y" com os valores que você escolher.

Mostrar solução
Python

estado = "RS"
print(f"Moro em {cidade}, {estado}")
🟢 Aquecimento

3. Crie uma variável para uma grandeza física de sua escolha (temperatura, pressão, velocidade…) e imprima-a com f-string, incluindo a unidade.

Mostrar solução
Python

print(f"A temperatura é {temperatura} °C")
🟢 Aquecimento

4. Crie uma variável nivel com valor 1, mostre-a, depois troque o valor para 2 e mostre novamente.

Mostrar solução
Python

print(nivel)
🟡 Praticando

5. Sem executar, tente adivinhar o tipo de cada valor: "42", 42 e 4.2. Depois confira com type().

Mostrar solução

"42" é texto (str), por causa das aspas; 42 é inteiro (int); 4.2 é decimal (float).

Python 
print(type("42"))
print(type(42))
print(type(4.2))
🟡 Praticando

6. Calcule e guarde a área de um retângulo de base 4 e altura 3 numa variável, depois mostre o resultado com f-string.

Mostrar solução
Python 
base = 4
altura = 3
area = base * altura
print(f"Área: {area}")
🟡 Praticando

7. Guarde um preço fixo numa variável e calcule o dobro dele numa segunda variável. Mostre o resultado com f-string.

Mostrar solução
Python

print(f"Dobro: {dobro}")
🟡 Praticando

8. Qual é o tipo do resultado de 2 + 3.0? Tente adivinhar antes de conferir com type().

Mostrar solução

Quando somamos um int com um float, Python devolve um float — o tipo mais "amplo" vence.

Python

Mostrar solução

print(f"Período = {T} s")

Prévia: aonde isso leva

Você ainda não precisa entender este código — execute e veja variáveis sendo usadas para montar um gráfico de barras, algo que você vai aprender a fazer mais adiante.

Python 
import matplotlib.pyplot as plt
plt.bar(["aço", "alumínio", "cobre"], [7850, 2700, 8960])
plt.title("Densidades guardadas (prévia)")
plt.ylabel("kg/m³")
plt.show()
