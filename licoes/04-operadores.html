Operadores

Operadores são os símbolos que usamos para fazer contas, comparar valores e combinar condições. Eles são a base de quase toda decisão que um programa toma.

⚡

Pense nos intertravamentos de segurança de uma máquina industrial: o motor só pode ligar se a proteção estiver fechada E a energia estiver presente. Faltou uma das condições? Motor bloqueado. Em Python, os operadores and, or e not funcionam exatamente assim: combinam condições para tomar decisões seguras.

Operadores aritméticos

São os das contas que você já conhece, com alguns extras:

  • + soma  ·  - subtração
  • * multiplicação  ·  / divisão
  • // divisão inteira (descarta a parte decimal)
  • % resto da divisão
  • ** potência (elevado a)
Python 
print(10 + 3)   # soma
print(10 / 3)   # divisão comum
print(10 // 3)  # divisão inteira
print(10 % 3)   # resto
print(2 ** 10)  # 2 elevado a 10
Tudo após # é comentário: o Python ignora. Servem para deixar anotações para humanos lerem.

O resto (%) é mais útil do que parece

O operador % devolve o que "sobra" de uma divisão. Ele é o jeito clássico de descobrir se um número é par: um número é par quando o resto da divisão por 2 é zero.

Python 
print(8 % 2)   # 0 → oito é par
print(7 % 2)   # 1 → sete é ímpar

Operadores de comparação

Comparam dois valores e respondem com True (verdadeiro) ou False (falso): um novo tipo de dado chamado booleano (bool).

  • == igual a  ·  != diferente de
  • > maior  ·  < menor
  • >= maior ou igual  ·  <= menor ou igual
Python 
idade = 16
print(idade >= 18)   # já é maior de idade?
print(idade == 16)   # tem exatamente 16?
print(idade != 16)   # é diferente de 16?
Cuidado: um = guarda um valor na variável; dois == comparam dois valores. Trocar um pelo outro é um erro clássico.

Operadores lógicos

Servem para combinar condições: and (e), or (ou) e not (não).

  • and: verdadeiro só se ambas forem verdadeiras.
  • or: verdadeiro se pelo menos uma for verdadeira.
  • not: inverte (verdadeiro vira falso e vice-versa).
Python 
idade = 20
tem_ingresso = True

print(idade >= 18 and tem_ingresso)  # pode entrar?
print(idade < 12 or idade > 60)      # tem desconto?
print(not tem_ingresso)              # está sem ingresso?

Na engenharia

Em engenharia, comparações e operadores lógicos aparecem em todo lugar: verificar tolerâncias de fabricação, faixas de operação segura, qualidade de sinal. O programa calcula e imprime diretamente um booleano: a resposta da máquina.

⚙️ Mecânica

Verificar se uma medida está dentro da tolerância de 10 ± 0,2 mm:

Python 
medida = 10.05
print("Dentro?", medida >= 9.8 and medida <= 10.2)
⚡ Elétrica

Verificar se a tensão está na faixa segura de operação (210–230 V):

Python 
tensao = 220
print("Seguro?", tensao >= 210 and tensao <= 230)
📡 Telecomunicações

Verificar se o sinal recebido está acima do limiar de –70 dBm:

Python 
sinal = -65
print("Link OK?", sinal > -70)

Prévia: aonde isso leva

Você ainda não precisa entender este código: execute e veja como booleanos de tolerância viram um gráfico com linhas-limite, algo que você vai explorar mais adiante.

Python 
import matplotlib.pyplot as plt
medidas = [9.7, 9.9, 10.0, 10.1, 10.3]
plt.plot([1, 2, 3, 4, 5], medidas, marker="o")
plt.axhline(9.8, color="red", linestyle="--")
plt.axhline(10.2, color="red", linestyle="--")
plt.title("Medidas vs. tolerância (prévia)")
plt.xlabel("amostra")
plt.ylabel("medida (mm)")
plt.grid(True)
plt.show()
