Operadores

Operadores são os símbolos que usamos para fazer contas, comparar valores e combinar condições. Eles são a base de quase toda decisão que um programa toma.

Operadores aritméticos

São os das contas que você já conhece, com alguns extras:

  • + soma  ·  - subtração
  • * multiplicação  ·  / divisão
  • // divisão inteira (descarta a parte decimal)
  • % resto da divisão
  • ** potência (elevado a)
Python 
print(10 + 3)   # soma
print(10 / 3)   # divisão comum
print(10 // 3)  # divisão inteira
print(10 % 3)   # resto
print(2 ** 10)  # 2 elevado a 10
Tudo após # é comentário: o Python ignora. Servem para deixar anotações para humanos lerem.

O resto (%) é mais útil do que parece

O operador % devolve o que "sobra" de uma divisão. Ele é o jeito clássico de descobrir se um número é par: um número é par quando o resto da divisão por 2 é zero.

Python 
print(8 % 2)   # 0 → oito é par
print(7 % 2)   # 1 → sete é ímpar

Operadores de comparação

Comparam dois valores e respondem com True (verdadeiro) ou False (falso) — um novo tipo de dado chamado booleano (bool).

  • == igual a  ·  != diferente de
  • > maior  ·  < menor
  • >= maior ou igual  ·  <= menor ou igual
Python 
idade = 16
print(idade >= 18)   # já é maior de idade?
print(idade == 16)   # tem exatamente 16?
print(idade != 16)   # é diferente de 16?
Cuidado: um = guarda um valor na variável; dois == comparam dois valores. Trocar um pelo outro é um erro clássico.

Operadores lógicos

Servem para combinar condições: and (e), or (ou) e not (não).

  • and: verdadeiro só se ambas forem verdadeiras.
  • or: verdadeiro se pelo menos uma for verdadeira.
  • not: inverte (verdadeiro vira falso e vice-versa).
Python 
idade = 20
tem_ingresso = True

print(idade >= 18 and tem_ingresso)  # pode entrar?
print(idade < 12 or idade > 60)      # tem desconto?
print(not tem_ingresso)              # está sem ingresso?

Exercícios

1. Peça um número à pessoa e mostre se ele é par (use % e mostre o resultado da comparação == 0).

Mostrar solução
Python 
n = int(input("Digite um número: "))
print("É par?", n % 2 == 0)

2. Uma loja dá desconto para quem gasta mais de R$ 100 e é cliente cadastrado. Crie as variáveis valor e cadastrado e mostre, com um operador lógico, se a pessoa tem direito ao desconto.

Mostrar solução
Python 
valor = 150
cadastrado = True
print("Tem desconto?", valor > 100 and cadastrado)
