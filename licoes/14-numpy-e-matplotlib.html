NumPy e Matplotlib para engenharia

⚡

NumPy é uma calculadora vetorial: em vez de tratar número por número num laço, ela opera no vetor inteiro de uma vez — como processar um sinal completo de medições numa única conta. O Matplotlib transforma esses vetores em gráficos.

Criando arrays com NumPy

Comece importando o NumPy com o apelido np (convenção universal). np.array() cria um array a partir de uma lista Python:

Python 
import numpy as np

a = np.array([1, 2, 3, 4])
print(a)
print(type(a))

Repare que o NumPy imprime arrays sem vírgulas: [1 2 3 4]. Isso diferencia arrays NumPy de listas Python.

Operações vetorizadas

Com NumPy, você aplica operações matemáticas ao array inteiro de uma vez — sem for:

Python 
import numpy as np

a = np.array([1, 2, 3, 4])
print(a * 2)
print(a + 10)

Gerando sequências: arange e linspace

np.arange(início, fim) gera inteiros como o range(), mas como array NumPy. np.linspace(início, fim, n) gera n pontos igualmente espaçados entre início e fim (inclusivo):

Python 
import numpy as np

print(np.arange(0, 10))
print(np.linspace(0, 1, 5))

Funções matemáticas sobre arrays

NumPy oferece funções como np.sin, np.exp que operam em cada elemento do array automaticamente:

Python 
import numpy as np

t = np.linspace(0, 1, 5)
print(np.sin(2 * np.pi * t))
print(np.exp(-t))

Agregações: média, desvio padrão, máximo

Os arrays NumPy têm métodos de agregação prontos — muito mais diretos que somar manualmente num laço:

Python 
import numpy as np

a = np.array([3, 9, 2, 7, 5])
print("média:", a.mean())
print("desvio padrão:", a.std())
print("máximo:", a.max())
Vetorização: a operação age no array todo de uma vez — mais limpo (e rápido) que um laço para muitos dados. Em vez de for x in lista: resultado.append(x * 2), escreva simplesmente array * 2.

Plotando com Matplotlib

Combine NumPy (para gerar os dados) com Matplotlib (para visualizá-los). plt.plot(x, y) recebe arrays diretamente:

Python 
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(0, 2 * np.pi, 100)
y = np.sin(x)
plt.plot(x, y)
plt.title("Senoide gerada com NumPy")
plt.xlabel("x")
plt.ylabel("sin(x)")
plt.grid(True)
plt.show()

Na engenharia

NumPy e Matplotlib aparecem em todas as engenharias para calcular e visualizar dados.

⚙️ Mecânica

Posições da queda livre vetorizadas: todas as alturas de uma vez, sem laço.

Python 
import numpy as np
t = np.linspace(0, 2, 5)
y = 0.5 * 9.81 * t**2
print(y)
⚡ Elétrica

Média e desvio padrão de tensões medidas numa fonte de 5 V:

Python 
import numpy as np
tensoes = np.array([4.9, 5.1, 5.0, 4.8, 5.2])
print(tensoes.mean(), tensoes.std())
📡 Telecomunicações

Sinal senoidal de 2 Hz amostrado em 5 pontos de 0 a 1 s:

Python 
import numpy as np
t = np.linspace(0, 1, 5)
s = np.sin(2 * np.pi * 2 * t)
print(s)

14. (⚡ Elétrica) Calcule a carga de um capacitor v = 5 × (1 − e−t/τ), τ = 1, t de 0 a 5 s com 100 pontos. Imprima a tensão final. Depois plote a curva com a assíntota em 5 V (linha tracejada vermelha).

Mostrar solução
Sem gráfico (texto)
Python 
import numpy as np
t = np.linspace(0, 5, 100)
v = 5 * (1 - np.exp(-t / 1.0))
print("tensão final:", v[-1])
Com gráfico (matplotlib)
Python 
import numpy as np
import matplotlib.pyplot as plt
t = np.linspace(0, 5, 100)
v = 5 * (1 - np.exp(-t / 1.0))
plt.plot(t, v, label="v(t)")
plt.axhline(5, color="red", linestyle="--", label="5 V")
plt.title("Carga do capacitor (NumPy)")
plt.xlabel("t (s)")
plt.ylabel("v (V)")
plt.legend()
plt.grid(True)
plt.show()

15. (📡 Geral) Some dois sinais: s1 = sin(2π·2·t) e s2 = sin(2π·5·t), com t de 0 a 1 s e 300 pontos. Imprima o máximo da soma. Depois plote os três sinais (s1, s2 e a soma) na mesma figura com legendas.

Mostrar solução
Sem gráfico (texto)

Com gráfico (matplotlib)
Python 
import numpy as np
import matplotlib.pyplot as plt

s1 = np.sin(2 * np.pi * 2 * t)
s2 = np.sin(2 * np.pi * 5 * t)

plt.plot(t, s1, label="2 Hz")
plt.plot(t, s2, label="5 Hz")
plt.plot(t, soma, label="soma")
plt.title("Soma de sinais")
plt.xlabel("t (s)")
plt.ylabel("amplitude")
plt.legend()
plt.grid(True)
plt.show()
