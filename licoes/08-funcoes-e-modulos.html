Funções e módulos

⚙️

Pense numa caixa de engrenagens: você conecta o eixo de entrada, a caixa processa (multiplica torque, reduz velocidade) e sai um valor pelo eixo de saída — sem precisar abrir a caixa toda vez. Uma função é a mesma ideia: você dá os parâmetros (entrada), ela executa o processamento e return entrega o resultado (saída). Escreva uma vez, reutilize onde quiser, e o nome da função documenta a intenção.

Definindo e chamando uma função

Para criar uma função use a palavra-chave def, dê um nome, liste os parâmetros entre parênteses e use return para devolver o resultado. Depois basta chamá-la pelo nome:

Python 
def saudacao(nome):
    return f"Olá, {nome}!"

print(saudacao("Engenharia"))

Por que usar funções?

Funções evitam repetição de código, facilitam testes e deixam o programa mais legível. O nome da função descreve o que ela faz; quem lê não precisa entender os detalhes internos para usá-la:

Python 
def area_retangulo(base, altura):
    return base * altura

print(area_retangulo(3, 4))
print(area_retangulo(10, 2.5))

Parâmetros e argumentos padrão

Um parâmetro pode ter um valor padrão: se você não passar aquele argumento na chamada, o padrão é usado automaticamente. Isso torna a função mais flexível sem exigir que o chamador sempre forneça todos os valores:

Python 
def potencia(base, expoente=2):
    return base ** expoente

print(potencia(5))     # usa expoente=2 por padrão → 25
print(potencia(2, 3))  # expoente=3 → 8
Variáveis locais: variáveis criadas dentro de uma função existem apenas ali — elas "nascem" quando a função é chamada e "desaparecem" quando ela retorna. Isso evita conflitos com variáveis de fora e é uma das grandes vantagens de encapsular código em funções.

Usando módulos prontos com import

Python vem com uma vasta biblioteca padrão de funções prontas, organizadas em módulos. O módulo math, por exemplo, traz raiz quadrada, logaritmo, seno, pi e muito mais. Basta importar:

Python 
import math

print(math.sqrt(16))      # raiz quadrada → 4.0
print(math.pi)            # π ≈ 3.14159…
print(math.log10(1000))   # log base 10 → 3.0

Na engenharia

Funções tornam fórmulas de engenharia reutilizáveis e com nome claro. Veja o mesmo padrão nas três áreas.

⚙️ Mecânica

Tensão normal σ = F/A — força dividida pela área da seção transversal:

Python 
def tensao_normal(forca, area):
    return forca / area

sigma = tensao_normal(5000, 0.02)
print(f"σ = {sigma:.1f} Pa")
⚡ Elétrica

Corrente elétrica pela Lei de Ohm — I = V/R:

Python 
def corrente(tensao, resistencia):
    return tensao / resistencia

I = corrente(12, 470)
print(f"I = {I*1000:.2f} mA")
📡 Telecomunicações

Ganho em dB de um amplificador — 20·log₁₀(saída/entrada):

Python 
import math

def ganho_db(saida, entrada):
    return 20 * math.log10(saida / entrada)

g = ganho_db(10, 1)
print(f"Ganho = {g:.1f} dB")

15. (⚙️ geral) Escreva a função aproxima_pi(n) usando a série de Leibniz (π/4 = 1 − 1/3 + 1/5 − …). Imprima a aproximação para n = 10, 100 e 1000 termos. Depois plote a convergência rumo a π.

Mostrar solução
Sem gráfico (texto)
Python 
def aproxima_pi(n):
    s = 0.0
    for k in range(n):
        s += (-1)**k / (2*k + 1)
    return 4 * s

for n in [10, 100, 1000]:
    print(f"n={n}: {aproxima_pi(n):.5f}")
Com gráfico (matplotlib)
Python 
import math
import matplotlib.pyplot as plt

def aproxima_pi(n):
    s = 0.0
    for k in range(n):
        s += (-1)**k / (2*k + 1)
    return 4 * s

ns = list(range(1, 201))
aprox = [aproxima_pi(n) for n in ns]
plt.plot(ns, aprox, label="aproximação")
plt.axhline(math.pi, color="red", linestyle="--", label="π")
plt.title("Série de Leibniz → π")
plt.xlabel("número de termos")
plt.ylabel("valor")
plt.legend()
plt.grid(True)
plt.show()
